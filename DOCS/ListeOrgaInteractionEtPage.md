Feature User :

-Poster un ticket
-Consulter leurs tickets (tous états compris, dernier actif)
-Chatter dans leurs tickets
-Envoyer des images dans leur tickets
-Consulter leur page de compte
-Consulter la page d'accueil
-Se déconnecter

Feature Admin :

-Consulter les tickets
-Consulter les tickets archivés
-Chatter dans les tickets
-Changer l'état d'un ticket
-Inviter un Admin ou Super Admin sur un ticket
-Consulter leur page de compte
-Changer l'état d'un compte (suspendre ou fermer)
-Consulter les invitations envoyées pour un ticket
-Consulter leurs invitations reçues pour un ticket
-Consulter la page d'accueil
-Se déconnecter

Feature Super Admin :

-Tous les droits Admin
-Créer et fermer des comptes Admin et User
-Créer et supprimer des catégories de ticket
-Se déconnecter
