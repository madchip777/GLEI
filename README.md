# Installation:

### Windows : 
- Installez WAMP;


- Et choisir MySQL, puis PHP 8.5 dans le menu en bas a droite.

### Debian :
- php, mysql


- activez Composer pour php `composer global require laravel/installer` (sois ce [ce tutoriel](https://laravel.com/docs/13.x/installation) )


- wip (lookup fedora atm)

### Fedora : 
- Pour PHP et MySQL, suivez [ce tutoriel](https://bmehul.com/articles/install-apache-php-and-mysql-on-fedora-43) (en replacent 8.4 par 8.5)


- Activez Composer pour PHP `composer global require laravel/installer`


- Puis suivez [ce tutoriel](https://laravel.com/docs/13.x/installation)
